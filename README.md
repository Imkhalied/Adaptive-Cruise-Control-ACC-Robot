# Adaptive Cruise Control (ACC) Robot using TinyML & PID Control

## Project Overview

Development of an intelligent Adaptive Cruise Control (ACC) robot that combines PID control and TinyML to maintain a safe following distance. The project integrates real-time sensor data, embedded AI, and autonomous speed control on an ESP32 platform.

Completed as the capstone project of the **Robotics & Artificial Intelligence Scholarship Program (90 Hours)** at EncryptCore.

---

## Project Contents

- Adaptive Cruise Control (ACC) robot development
- ESP32 embedded control system
- HC-SR04 ultrasonic distance sensing
- MPU6050 IMU integration
- MATLAB PID controller design and tuning
- Edge Impulse TinyML model training
- Real-world dataset collection and labeling
- AI deployment on ESP32 using the Edge Impulse C++ library
- Hardware design and wiring using Fritzing
